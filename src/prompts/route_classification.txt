- uds is for historical analytics; if the query asks for trend/last month/aggregated metrics, prefer uds.
- Definition-style questions about terms like FBA/FBM (e.g. "what is FBA") should be classified as ic_docs, not sp_api.

Return ONLY a single JSON object on one line:
- {"workflow": "<one of general|amazon_docs|ic_docs|sp_api|uds>", "confidence": <float between 0 and 1>}
- Do not include any explanations.
